# pi_web1

Biblioteca Virtual

Una aplicación web que permite a los usuarios explorar, buscar, registrar y administrar libros de manera digital. Diseñada para facilitar el acceso a la lectura y la gestión de bibliotecas desde cualquier lugar.

🚀 Características principales

🗂️ Catálogo de libros con detalles como título, autor, género y año.

🔍 Búsqueda avanzada por título, autor o categoría.

👤 Registro e inicio de sesión de usuarios.

📥 Préstamo y devolución de libros (modo virtual).

⭐ Valoraciones y comentarios por parte de los usuarios.

📚 Administración de inventario para bibliotecarios (panel de control).
